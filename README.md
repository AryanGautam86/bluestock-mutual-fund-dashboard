#  Bluestock Mutual Fund Dashboard

A complete interactive Mutual Fund Analytics Dashboard built using **Python**, **SQLite**, **Pandas**, **Plotly**, and **Streamlit**.

This dashboard helps users analyze mutual fund performance, NAV trends, investor behavior, and portfolio insights through an intuitive interface.

---

##  Features

###  Dashboard Overview
- Summary KPIs
- Total Mutual Funds
- Fund Houses
- Investors
- NAV Records
- Total Transactions
- Average Returns
- Total Investment

---

###  NAV Analysis
- Filter by Mutual Fund
- Interactive NAV Trend Charts
- NAV Data Table
- Download Filtered CSV

---

###  Performance Analysis
- Fund Performance Comparison
- Return Analysis
- Performance Visualizations
- Download Filtered Data

---

###  Investor Analysis
- Investor Statistics
- State-wise Analysis
- Gender Distribution
- Occupation Distribution
- Investment Distribution
- Download CSV

---

###  Portfolio Analysis
- Filter Portfolio by Fund
- Investment Distribution
- Portfolio Summary
- Download Portfolio Data

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AryanGautam86/bluestock-mutual-fund-dashboard.git
```

Go inside the project

```bash
cd bluestock-mutual-fund-dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run dashboard/app.py
```
